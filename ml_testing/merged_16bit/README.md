# medical-translator-es-en : GGUF

This model was finetuned and converted to GGUF format using [Unsloth](https://github.com/unslothai/unsloth).

**Example usage**:
- For text only LLMs:    `llama-cli -hf dammi19/medical-translator-es-en --jinja`
- For multimodal models: `llama-mtmd-cli -hf dammi19/medical-translator-es-en --jinja`

## Available Model files:
- `llama-3.2-3b-instruct.Q4_K_M.gguf`
